Internet

The Internet is a global network of computers.

Web

The Web is huge system of hypertext documents that can be accessed through the internet. It is used to view webpages and websites through web browsers.

How are the Internet and Web different?

The internet and the web are different because the internet is the global network, while the web is the information on that network. Basically the internet is the hardware and the web is the software.

ASCII and Unicode

ASCII stands for American Standard Code for Information Interchange. It is a standard for encoding English characters to unique numbers, allowing computers to communicate with each other.

Unicode is similar to ASCII, but is a universally standardized system that encodes a unique numerical value to every characater in many languages around the world.

What is UTF-8?

UTF-8 is a transformation format to take Unicode and represent it in a computer readable format, using 8-bit bytes. It is the standard used on the web, allowing internalization, making text be shown correctly in many different languages.

URL

URL stands for uniform resource locator, and is the address of a web page.

Domain Name

Domain name is the part of the website address that tells which website it is for. An example is 'www.google.com', where google is the domain name.

HTTP

HTTP stands for hypertext transfer protocol, and is used to load webpages, transferring information between the client or webpage and a server.

HTTPS

HTTPS stands for hypertext transfer protocol secure. It is used similar to HTTP, but is encrypted.

IP Address

An IP address is a unique string for each deivce communicating over the internet. It allows specific computers to identify each other and communicate, allowing data transfer between them. IP addresses are made up of numbers like '8.8.8.8'.

DNS

DNS stands for domain name system. It translates human readable domain names like www.google.com into computer readable IP addresses.

W3C Recommendations

W3C stands for World Wide Wed Consortium. They make recommendations for html for interoperability and accessibility between devices and users. and some them are: official specifictions for HTML versions, accessibility guidelines, best practices for code quality, and using semantic HTML for content. All of this helps with interoperability and helps things be consistent across the web and when using different browsers.
