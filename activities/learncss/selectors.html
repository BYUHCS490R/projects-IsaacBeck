You can select an item using tags, tag name, and descendant.

The difference between an ID and a class is there can only be one id, but class can be used many times.

The difference between a SPAN tag and a DIV tag is that while they both select stuff, a DIV tag causes line breaks, and a SPAN tag does not.

The difference between a DIV tag and a HEADER, MAIN, or FOOTER is that there is no difference, they are also DIV tags, they group things together.

The different ways to select a color are using a name, or rgb.
