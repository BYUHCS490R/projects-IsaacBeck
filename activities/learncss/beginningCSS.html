Beginning CSS

Inline CSS is css within a line. Embedded CSS is css in the header of the file. External CSS is css in another file as a .css file.

You should only use external CSS because of readability, organization, and updating all webpages on a website more easily by changing one file instead of needing to change all the inline or embedded css individually.

A selector is used to change some tags instead of all of them, or a specific tag.
