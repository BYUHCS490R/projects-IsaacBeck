Box Model

The difference between block tags and inline tags is that block tags are a box, by default taking up the width of the screen. Inline tags are things like span tags, which are inside a line.

Padding is space on the inside of a box, and margin is the outside of the box.

The units of measurement you can use in CSS are px, pt, pc, in, em, and rem.

You would use a figure tag to make your content change for various screen sizes.
