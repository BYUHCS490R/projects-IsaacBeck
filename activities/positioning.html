Positioning

Normal flow in html is the default way the html elements are displayed before any CSS.

Float makes the text flow around some content. You need to clear the float to make sidebars.

Absolute makes elements stay in one place. Fixed makes an element be fixed relative to the viewport. Relative makes you able to slide an element around.

You can make nice navigation buttons for mobile devices by using block tags and padding.

You can find videos to review what we talked about in class in the modules on Canvas.
