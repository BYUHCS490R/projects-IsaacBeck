The viewport meta tag is important because it makes the page responsive.

You can adjust the CSS for printing by making a print css page using a media query, which is what is printed when you print something.

You would want to adjust the CSS for printing only when we know that people will print our page.

We need the viewport meta tag for the page to work on a mobile device.

If you select the same tag twice, and in one of them you say “flex-direction: column;” and the other says “flex-direction: row;”, the last one takes priority.

You change the CSS for a web page displayed on a phone by using media queries.
