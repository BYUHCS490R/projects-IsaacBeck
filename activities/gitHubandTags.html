2. Engagements are due at midnight, and can't be turned in late, but four are dropped.

4. The tags are: DOCTYPE, html, head, meta, title, body, h1, and p.

DOCTYPE is which version is being used.

html is a root element, with the language set to english.

head is the container for the title.

meta is the encoding used, UFT-8, and the viewport is for the size of the screen.

title is what the title is.

body is the main body of text.

h1 is a heading tag, to make a header for the text.

p is a paragraph tag, to make the main body of text.
