The form tag has an "action" attribute which specifies where to send the form data when a form is submitted.

The form tag has a "method" attribute which does POST or GET.

The difference between a "POST" and a "GET" is that GET has the data in the URL, while POST does not. POST is changing (adding, updating, or deleting) data, while GET is reading.

The different ways to style your form are by using spacing, alignment, colors, and box-shadow.

Flex helps for both Desktop and Mobile phones by using media queries.
